Small boats API

I noticed that the UK government website keeps a live updated table of refugee small boat arrivals for the last 7 days. There wasn’t an API for this data and I thought it could use one. Using Val Town I created an API that fetches this data so it can be easily more easily accessed and immplemented into research projects.
