Apple Notes

Apple notes is such a relief to use. I feel the urge to write this because I’m honestly blown away by a default tool. I had ignored it for so long since it’s just the basic tool that comes with the operating system, a tool which you put aside as soon as you can to download something better.

It achieved everything I wanted a note taking tool to do in the simplest possible way. When I started using an iPad for design work and sketching I gave Notes a try as I was aware that Apple made it easy for syncing of information across devices. If I made sketches on the iPad I wanted them to appear on the laptop as reference for more technical design work without any hassle.

The first thing I felt blessed by was the absolute lack of any nonsense tools and features that I just wanted out of the way. There wasn’t a login screen, settings to customise. It wasn’t a web app. A clean white page for text and a basic menu for note folders. Other than several minimal icons such as to add a tick box or a table, that was basically it.

I wonder if the fact that it was made by Apple and it comes with their devices they didn’t feel it necessary to fill it up with dozens of features. It’s just a tool, that’s tucked away, works and is there when you need it. Much love to you.
